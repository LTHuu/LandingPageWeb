# FITVII Health & Fitness Tracker Landing Page

A modern landing page for a smartwatch product built with Next.js and ASP.NET Core Web API.

## Features

- Responsive Landing Page
- FAQ loaded from PostgreSQL
- Contact/Question submission form
- RESTful API
- Docker support
- Cloud deployment

## Tech Stack

### Frontend
- Next.js 16
- React
- TypeScript
- Tailwind CSS

### Backend
- ASP.NET Core 9
- Entity Framework Core
- PostgreSQL
- Docker

### Deployment
- Frontend: Vercel
- Backend: Render

## Project Structure

```
LandingPageWeb
│
├── frontend/
│   └── Next.js
│
└── backend/
    └── ASP.NET Core Web API
```

## API

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | /api/FAQ/getAll | Get all FAQs |
| POST | /api/Submit/create | Submit customer question |
